模块使用

1. 下载克隆模块

```
git clone https://github.com/Mehaei/AmazonCaptcha.git
```

2. 使用模块

```
from CaptCha import AmazonVerifyCode
amazon_verify = AmazonVerifyCode()
img_path = "/User/test.jpg"
# 返回验证码
amazon_verify.get_verify_code(img_path)
```
