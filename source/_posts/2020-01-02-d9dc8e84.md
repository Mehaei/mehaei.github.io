# 安装hdparm

```
sudo apt-get install hdparm
```

# 查看磁盘

```
df -lh
```

# 测试读速度

```
sudo hdparm -t /dev/sda
```

# 测试写速度

```
sudo time dd if=/dev/zero bs=1024 count=1000000 of=/1Gb.file
```
