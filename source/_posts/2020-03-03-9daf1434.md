# 找不到--mount命令

报错详情:

```
Error response from daemon: Dockerfile parse error line 108: Unknown flag: mount
```

解决:

在dokcerfile文件中 在第一行加入:

```
# syntax=docker/dockerfile:experimental
```

启动:

```
DOCKER_BUILDKIT=1 docker build -f Dockerfile -t test_name .
```
